# Прымусовая праца ў месцах зняволення Беларусі — інтэрактыўная мапа

Гэта мінімальная рабочая версія сайта для GitHub Pages.

## Файлы

- `index.html` — галоўная старонка з мапай.
- `data.csv` — даныя па ўстановах, каардынатах і заробках.

## Як абнаўляць даныя

1. Адкрыйце `data.csv`.
2. Змяніце або дадайце радок.
3. Захавайце файл.
4. Загрузіце абноўлены файл у GitHub.

## Важна па бяспецы

Не публікуйце імёны рэспандэнтаў, поўныя цытаты, якія могуць ідэнтыфікаваць чалавека, дакладныя даты этапавання, калі яны не патрэбныя для публічнай версіі.

## Публікацыя

Сайт можна апублікаваць праз GitHub Pages: Settings → Pages → Deploy from a branch → main → /root.
