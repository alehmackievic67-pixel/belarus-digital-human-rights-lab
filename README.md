# Прымусовая праца ў месцах зняволення Беларусі

Доўгатэрміновы стартавы каркас інтэрактыўнай платформы для GitHub Pages.

## Структура

- `index.html` — галоўная старонка.
- `css/style.css` — візуальнае афармленне.
- `js/map.js` — загрузка даных, мапа, маркеры, табліца.
- `js/filters.js` — фільтры.
- `data/prisons.json` — публічныя даныя па ўстановах.
- `data/testimonies.json` — будучыя ананімізаваныя сведчанні.
- `data/wages.json` — будучая асобная база заробкаў.
- `data/injuries.json` — будучая база траўмаў і бяспекі працы.
- `data/food.json` — будучая база харчавання.
- `images/` — выявы і інфаграфіка.
- `reports/` — публічныя справаздачы.
- `prison/` — будучыя асобныя старонкі калоній.

## Бяспека

Не публікаваць імёны, кантакты, поўныя цытаты, дакладныя даты этапавання, камбінацыі фактаў, якія могуць ідэнтыфікаваць рэспандэнтаў.
